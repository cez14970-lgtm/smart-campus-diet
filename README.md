**AI-powered personalized campus diet assistant for CSU students.**

面向中南大学学生的 AI 个性化校园健康饮食服务平台。通过智能健康画像分析，从食堂真实菜品中为你推荐最适合的营养搭配方案。

> 本项目为创新创业项目技术展示原型（MVP），所有数据均为 Mock 数据，不涉及真实用户隐私和第三方 API。

---

## 核心功能

### 1. 个人健康画像
填写年龄、性别、身高、体重、运动频率、饮食目标（健身增肌/减脂控重/熬夜恢复等）、过敏忌口、近期状态等，构建专属健康画像。

### 2. AI 饮食建议
基于规则引擎分析用户画像，生成：
- 今日饮食重点
- 推荐补充的食材类型
- 需要减少的饮食类型
- 风险提醒
- 自然语言风格的 AI 建议文本
- 健康营养雷达图

### 3. 食堂菜品推荐
- 覆盖中南大学二食堂、三食堂、南校区食堂、铁道校区食堂
- 30+ 真实可购菜品，标注食材、标签、过敏原
- 智能推荐 3 套可选套餐（最优推荐/经济实惠/多样选择）
- 每套套餐包含主食+蛋白质+蔬菜+汤/饮品

### 4. 健康餐订餐流程
- 选择推荐套餐，模拟下单
- 自定义取餐食堂、取餐时间、是否配送
- 实时计算预估价格
- 下单成功展示订单卡片

### 5. 校园饮食社区
- **周边红黑榜**：8 家校园周边餐厅，含健康评分、口味评分、红/黑榜标识和推荐理由
- **饮食论坛**：学生帖子 mock 数据展示，话题涵盖健身饮食、考试周饮食、周边测评等

### 6. 技术架构说明
展示系统架构总览、推荐引擎流程、技术栈选型、后期扩展计划、API 预留接口说明。

## 本地运行

### 前置要求
- Node.js >= 18
- npm >= 9

### 安装与启动

```bash
# 克隆项目
git clone <your-repo-url>
cd smart-campus-diet

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

开发服务器默认运行在 `http://localhost:5173`。

## 推荐逻辑说明

推荐引擎位于 [src/lib/recommendation.ts](src/lib/recommendation.ts)，核心流程：

1. **BMI / 代谢计算**：基于身高体重计算 BMI 和基础代谢率
2. **饮食目标解析**：将用户选择的饮食目标映射为推荐食材和避免食物列表
3. **过敏原过滤**：直接排除含过敏原的菜品
4. **菜品标签匹配与加权评分**：根据目标和状态为每个菜品打分
5. **套餐智能组合**：从高评分菜品中生成 3 套不同策略的套餐（最优/经济/多样）
6. **自然语言解释生成**：将推荐结果组装为类 AI 的自然语言建议文本

示例规则：
- **健身增肌** → 高蛋白标签加权，推荐鸡胸肉/牛肉/鸡蛋/鱼，避开油炸高糖
- **减脂控重** → 低卡高饱腹加权，推荐粗粮/蔬菜/白肉，避开油炸高GI
- **胃部不适** → 避免辛辣/油炸/过酸，推荐小米粥/鸡蛋羹/蒸南瓜/清蒸鱼
- **熬夜恢复** → 推荐清淡蛋白/绿叶蔬菜/含维生素B族食材

## 技术路线说明

### 当前 MVP 架构
采用 **规则引擎优先** 的混合方案：
- 推荐准确性由人工设计的营养学规则保证
- 输出 100% 可预测、可审计
- 无需训练模型、无需 API Key
- 快速验证产品核心逻辑

### 后期可扩展方向
本项目为以下真实 AI 能力预留了标准接口：

1. **向量数据库 + RAG**：将营养学知识库向量化，通过语义检索增强推荐质量
2. **LLM 解释生成**：接入 Claude/GPT 等大模型，生成更自然、更个性化的饮食建议
3. **LoRA 意图识别**：微调模型识别中南大学学生特有的饮食需求和偏好
4. **真实菜单数据**：对接食堂管理系统 API，获取实时菜品和价格
5. **订单系统**：打通真实点餐和支付流程
6. **小程序版本**：适配微信小程序生态

详见 [技术架构页面](http://localhost:5173/tech) 的详细说明。

## 安全与隐私声明

- **数据存储**：所有用户填写的健康画像数据仅保存在浏览器 localStorage 中，不上传至任何服务器
- **无第三方追踪**：本项目不包含任何用户行为追踪、数据统计或第三方 SDK
- **无真实 AI API**：当前版本不使用任何外部 AI API，所有推荐由本地规则引擎完成
- **医疗免责**：
  > 本平台提供的是**饮食建议与生活参考**，不构成医疗诊断、治疗方案或营养处方。
  > BMI 数值为健康生活参考指标。如有特殊健康状况、过敏史、慢性疾病或需要专业饮食方案，请务必咨询专业医生或注册营养师。
  > 涉及病史、过敏信息时，请在填写前确认信息准确性，并在实际点餐时向食堂工作人员确认菜品配料。
